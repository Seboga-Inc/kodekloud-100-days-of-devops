# Day 03 Secure Root SSH Access

## What I Did
- Commands practiced:
  - `sudo nano /etc/ssh/sshd_config`
  - `PermitRootLogin no`
  - `sudo systemctl restart sshd`

## What I Learned
- Go to /etc/ssh/sshd_config with sudo and nano -> look for the PermitRootlogin set it to yes or no
- Restart the system
